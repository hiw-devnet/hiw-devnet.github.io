HTMLとは

HTML (HyperText Markup Language) は複数の文書を相互に関連付け、結び付けるハイパーテキストを記述するためのマークアップ言語の一つです。主にWebページを作成する際に使用され、CSS (Cascading Style Sheets) を使用してスタイルを設定したり、スクリプト (主に JavaScript) を利用することができます。

以下のコードは最初からコードを記述する際のテンプレートで、更に下で解説を行います。

<!DOCTYPE html>
<html lang="ja">
    <head>
        <meta charset="utf-8">
        <title></title>
        <style>

        </style>
        <script>

        </script>
    </head>
    <body>

    </body>
</html>

基本用語
<>の中に記述されたものをタグといいます。内側に/が含まれたものがあり、それを終了タグといいます。また、/ がない方 (開始タグ) 側に名前="値"の構文で属性を指定できます。

<img>
画像タグです。src=""属性に画像URL (パス、拡張子込み) を指定して画像を表示します。またwidth=""属性に数値 (単位無しまたは%)で横幅、height=""属性を同様に指定して高さを指定することができます。

ただし、これには注意が必要で、ブラウザによって表示できる拡張子が異なります。一般的な.jpgや.png, .webpなどは、ほぼすべてのブラウザで表示できますが、WebPよりさらに圧縮率が高い.avifはiOSのWebKitベースブラウザ (Safari や Chrome)などでは表示できない他、同様の拡張子が存在します。

また、ベクター画像としてもっとも一般的なSVGにも注意が必要です。この拡張子はXML形式で記述されており、Inkscapeなどの描画ソフトで作成した場合、font-faceが保存されない問題があります。これによって端末に搭載されていないフォントが表示できず、ブラウザによって選択される可能性が高くなります。
<a>
アンカータグです。href=""属性に URL (パス) を指定して他のWebページやファイル、メールアドレスへのリンクを作成できます。

テンプレートの利用
上記のテンプレートを利用するには環境を確認する必要があります。まず文字コードを設定します。HTMLを記述するのに特に要件はなく、デフォルトで搭載されているメモ帳でも作成できますが、文字コードを指定しないと環境によっては文字化けする場合があります。

これを設定するには<meta>タグのcharset=""にコードを記述します。Windows メモ帳や Visual Studio Code, Atom などほとんどのテキストエディターではUTF-8が使用されているため編集する必要はありません。TeraPad などの古いテキストエディターでは Shif-JISが使用されていることもあり、その場合は文字コードの設定からUTF-8を指定しましょう。

また、CSS や JavaScript をインラインに記述するか、別のファイルとして読み込むかは状況によって異なります。例えば、ヘッダーやフッターなど複数のファイルでスタイルを流用する場合は別のファイルを読み込んだ方が良いでしょう。それ以外の場合はインラインへの記述で十分です。実際に外部ファイルを読み込むには下記のコードのsrc="", href=""それぞれ当てはまる部分にファイルのパスを指定してください。
<!-- CSSスタイルシートの読み込み -->
<link rel="stylesheet" href="https://example.com/stylesheet.css">
<!-- スクリプトの読み込み -->
<script src="https://example.com/script.js"></script>
